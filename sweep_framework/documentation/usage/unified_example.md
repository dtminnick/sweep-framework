# Unified Example

This section demonstrates how to use the framework end-to-end:

1. Define a `ModelConfig`
2. Load and split a dataset
3. Build loaders with a tokenizer
4. Run training with `ModelRun`
5. Analyze results with `SweepAnalyzer`
6. Export reports with `ReportBuilder`
